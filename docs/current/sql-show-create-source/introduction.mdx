---
title: "SHOW CREATE SOURCE"
description: "Use the `SHOW CREATE SOURCE` command to see the SQL statement used to create the specified source. By using this command, you can verify the source's settings and troubleshoot any issues."
---

## Syntax

```sql
SHOW CREATE SOURCE source_name;
```

## Parameters

| Parameter      | Description                                                            |
| -------------- | ---------------------------------------------------------------------- |
| _source\_name_ | The source for which you want to show the corresponding SQL statement. |
