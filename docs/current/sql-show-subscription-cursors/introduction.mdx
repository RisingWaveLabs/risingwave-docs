---
title: "SHOW SUBSCRIPTION CURSORS"
description: "Use the `SHOW SUBSCRIPTION CURSORS` command to display all subscription cursors in the current session."
---

## Syntax

```bash
SHOW SUBSCRIPTION CURSORS;
```

## Examples

```bash
SHOW SUBSCRIPTION CURSORS;

------RESULT
 Name | SubscriptionName
------+------------------
 cur2 | sub
 cur  | sub
(2 rows)
```
