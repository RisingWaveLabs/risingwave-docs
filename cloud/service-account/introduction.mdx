title: "Set up service accounts for applications"
sidebarTitle: Service account & API key
description: "In RisingWave Cloud, a user account typically represents an individual user who interacts with resources via the user interface. On the other hand, a service account represents an application. It accesses RisingWave Cloud resources on the application's behalf, using API keys for authentication."
---

This article will guide you through the process of setting up a service account for your applications and generating an API key.

## Create a service account

**Steps:**

1. Go to the [**Org.**](https://cloud.risingwave.com/organization/) tab and select **Service accounts**.
2. Click **New service account**.
<Frame>
  <img src="/images/cloud/service-account/org-create-service-account-d2853594bed75e4172ba00be4487baa6.png"/>
</Frame>
3. Enter a descriptive name for your service account and write a short description.
4. Click **Confirm**.
<Frame>
  <img src="/images/cloud/service-account/org-service-account-00bc8c372af54ffc1ca9bb4633335e34.png"/>
</Frame>

## Generate an API key

You can generate multiple API keys for a service account. Each API key will have a unique key ID and secret.

**Steps:**

1. In the account list, click the name of the service account you want to generate API keys for.
<Frame>
  <img src="/images/cloud/service-account/org-service-account-1-ad82e2e7f822c5dc55a0ada37bcea9b9.png"/>
</Frame>

1. Select the **API Keys** tab.
<Frame>
  <img src="/images/cloud/service-account/org-service-account-api-empty-de83ffe78972595f03ab0475a4195540.png"/>
</Frame>
1. Click **Add new API Key**.
2. Enter a descriptive name for your API key and click **Submit**.
3. A new API key will be generated. Please save the secret along with the ID to a safe place.

<Warning>
**CAUTION**

The secret will be shown only once. After you close the dialog, you won't be able to retrieve it. If you lose the secret, you have to generate a new API key.
</Warning>
<Frame>
  <img src="/images/cloud/service-account/org-service-account-api-key-2f31b4e49ec88248325bd21a071dece1.png"/>
</Frame>
