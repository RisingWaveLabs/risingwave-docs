---
title: "SHOW CREATE SINK"
description: "Use the `SHOW CREATE SINK` command to see the SQL statement used to create the specified sink. By using this command, you can verify the sink's settings and troubleshoot any issues."
---

## Syntax

```sql
SHOW CREATE SINK sink_name;
```

## Parameters

| Parameter    | Description                                                          |
| ------------ | -------------------------------------------------------------------- |
| _sink\_name_ | The sink for which you want to show the corresponding SQL statement. |
